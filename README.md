# Root-Cause-street-tree-project

## Files
- analysis.ipynb: main notebook
- filtered_data.csv: cleaned dataset
- 2015_Street_Tree_Census_-_Tree_Data_20260227.csv: raw dataset
- sidewalk_damage_model.pkl: trained model

## How to run
1. Install Python 3
2. Create environment and install packages:
   pip install pandas numpy matplotlib scikit-learn jupyter

3. Start notebook:
   jupyter notebook

4. Open `analysis.ipynb` and run cells from top to bottom.
